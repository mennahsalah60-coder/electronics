    <h1>Product Store</h1>

    <h2>Description</h2>
    <p>
        This project is a web page that displays electronic products with the ability to:
    </p>
    <ul>
        <li>View all products with images and prices.</li>
        <li>Add and remove products from the <strong>Cart</strong> with a counter showing the number of items.</li>
        <li>Add and remove products from the <strong>Wishlist</strong> with a wishlist counter.</li>
        <li>Search products by name with input validation.</li>
        <li>Filter products by price range with input validation.</li>
        <li>Display clear messages when no data is found or input is invalid.</li>
    </ul>

    <h3>3. Wishlist</h3>
    <ul>
        <li>Click the heart icon to add/remove products to/from the wishlist.</li>
        <li>Wishlist counter updates automatically and stays fixed on the page.</li>
        <li>Click the <strong>View Wishlist</strong> button to show only favorite products.</li>
    </ul>

    <h3>4. Search</h3>
    <ul>
        <li>Search by product name.</li>
        <li>Input validation: cannot be empty or whitespace only.</li>
        <li>Displays message when no results are found:
            <pre>&lt;p class="text-center fs-2 text-secondary"&gt;No Data Found&lt;/p&gt;</pre>
        </li>
    </ul>

    <h3>5. Price Filter</h3>
    <ul>
        <li>Enter minimum and maximum price to filter products.</li>
        <li>Input validation:
            <ul>
                <li>Both fields must be filled.</li>
                <li>Values must be positive numbers.</li>
                <li>Minimum must be less than or equal to maximum.</li>
            </ul>
        </li>
        <li>Displays message when no products are found:
            <pre>&lt;p class="text-center fs-2 text-secondary"&gt;No Data Found&lt;/p&gt;</pre>
        </li>
    </ul>

    <h2>Technologies Used</h2>
    <ul>
        <li>HTML</li>
        <li>CSS+ Bootstrap 5</li>
        <li>JavaScript (ES6)</li>
        <li>DOM Manipulation</li>
    </ul>
